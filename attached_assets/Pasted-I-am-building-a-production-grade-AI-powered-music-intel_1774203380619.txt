GLOBAL OBJECTIVE
================

Build a system that can:

1. Ingest audio (full mix OR vocal-only).
2. Extract deep musical intelligence:

   * Tempo, beat grid, downbeats
   * Time signature
   * Key, mode, modulations
   * Chords (advanced harmony)
   * Melody (note-level)
   * Bass movement
   * Structure (sections)
   * Dynamics, energy, groove

4. Generate full professional arrangements.
5. Orchestrate using high-end instrument modeling.
6. Apply style (genre + arranger profile).
7. Render final production-quality audio.
8. Export:

   * MIDI
   * MusicXML
   * PDF (notation)
   * Stems
   * Final audio (WAV/FLAC/MP3)
9. Provide a premium DAW-like GUI.

====================================================
MODULE SPECIFICATIONS
=====================

1. AUDIO INGESTION

Input:

* wav, mp3, flac, aiff, m4a

Process:

* resample
* normalize loudness (LUFS)
* trim silence
* generate waveform + spectrogram

Output:

* canonical audio
* metadata JSON

---

2. SOURCE SEPARATION

Use:

* Demucs (latest version)

Output:

* vocals.wav
* drums.wav
* bass.wav
* other.wav

Add:

* quality scoring
* fallback logic

---

3. RHYTHM ENGINE

Use:

* madmom
* librosa onset detection

Output:

* BPM
* beat grid
* downbeats
* time signature
* tempo map

---

4. KEY & MODE ENGINE

Use:

* HPCP/chroma (Essentia + librosa)
* sliding window analysis

Output:

* global key
* local key changes
* modulation timeline

---

5. CHORD ENGINE (HIGH PRIORITY)

Build hybrid model:

Features:

* chroma
* bass line
* meter alignment

Model:

* Transformer-based sequence model (SOTA)
* fallback HMM

Post-process:

* music21

Output:

* chord timeline
* Roman numerals
* lead sheet
* alternatives with confidence

---

6. VOCAL ENGINE

Use:

* torchcrepe (F0)
* Basic Pitch

Detect:

* pitch
* notes
* vibrato
* phrasing

Build model:

* melody → chord inference model

Output:

* melody MIDI
* inferred harmony (top 3 options)

---

7. STRUCTURE ENGINE

* novelty detection

Output:

* sections (intro, verse, chorus, etc.)
